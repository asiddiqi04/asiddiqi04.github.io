A couple of things I’m proud of. Open‑source where possible, always learning.

GreenScore — Hackathon Project

Open Demo / Case Study

Barcode scanner + search that estimates ethical & sustainability scores for consumer products using a RAG pipeline and AI‑assisted scoring. Built with friends during a hackathon.

  • React frontend with modern UI + Tailwind; FastAPI backend.
  • Data sources: UPCItemDB & Open Food Facts; MongoDB persistence.
  • AI: Gemini for score rationales; vector search for retrieval.
  • Features: scan/search history, favourites, and an in‑app AI helper.
